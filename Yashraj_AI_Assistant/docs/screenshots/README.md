# Screenshot Placeholders

This folder contains placeholders and guidance for screenshots to include in the GitHub README or project pages.

Required screenshots (place image files here):

- `home-page.png` - Home / landing page of the frontend
- `dashboard.png` - Dashboard view with analytics
- `chat-interface.png` - Chat interface demonstrating an assistant response
- `analytics-page.png` - Analytics / charts page
- `login-page.png` - Login or authentication page
- `api-docs.png` - FastAPI `/docs` OpenAPI documentation

You can replace these placeholders with real screenshots from a local run.
